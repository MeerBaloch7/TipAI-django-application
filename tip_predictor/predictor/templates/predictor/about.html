About Tip Predictor

How It Works

Our Tip Predictor uses a sophisticated XGBoost machine learning model trained on real restaurant data to provide accurate tip predictions.

Model Features

  • Total Bill Amount: The primary factor influencing tip amount
  • Party Size: Number of people dining together
  • Gender: Customer demographic information
  • Smoking Status: Whether the customer smokes
  • Day of Week: Dining patterns vary by day
  • Time of Day: Lunch vs. dinner service differences

Technology Stack

  • Python
  • Django Framework
  • XGBoost ML
  • Bootstrap CSS
  • Scikit-learn
  • SQLite
Model Performance

Our XGBoost model achieves high accuracy in predicting tip amounts based on comprehensive feature analysis.

Key Metrics:
• Low Mean Absolute Error
• High R² Score
• Robust Predictions
